01

Import into Tonic Studio

Import the complete library

In Tonic Studio, choose import from URL and paste the library address. Tonic Studio will read every hymn URL in the index.

https://khrihfahlabu.mualcin.com/index.json

Import one hymn

Open a hymn from the catalog, copy its JSON address, and paste that address into Tonic Studio’s URL importer.

https://khrihfahlabu.mualcin.com/hymns/1-pathian-cu-thangthat-si-ko-seh.json

Remote URL import is planned for Tonic Studio. Until it is available, download an individual JSON file and use Tonic Studio’s file importer.
